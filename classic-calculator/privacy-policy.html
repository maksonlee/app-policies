Privacy Policy

Classic Calculator Privacy Policy

Effective date: May 24, 2026

Classic Calculator is developed by Makson Lee. This Privacy Policy explains how Classic Calculator handles user data.

1. Core functionality

Classic Calculator provides basic calculator functions, including arithmetic operations, decimals, percent, backspace, memory functions, and result sharing.

2. Local processing

Calculator input, results, memory values, and display state are processed locally on the user's device. Classic Calculator does not upload calculator input or results to developer-controlled servers.

3. Data collection

Classic Calculator does not require user accounts and does not collect personal information such as name, email address, phone number, precise location, payment information, or contacts.

Classic Calculator does not collect analytics tied to a user identity through developer-controlled servers.

4. Permissions and device access

Classic Calculator may use the following device capability:

  • Vibration, so button presses can provide haptic feedback

This capability is used only to provide calculator interaction feedback.

5. Sharing

If the user taps the share button, Android opens the system share sheet so the user can choose another app to share the displayed result.

Classic Calculator does not control how the selected third-party app handles shared content.

6. Data sharing

Classic Calculator does not sell, rent, or share personal data with third parties.

The app may rely on standard Android and Google Play platform components to run on the device, but Classic Calculator does not transmit calculator input or results to developer-operated services.

7. Data retention and deletion

Because calculator processing occurs locally, Classic Calculator does not retain user data on developer-controlled servers.

If the app stores any temporary operational state on the device, that data remains on the user's device and can be removed by clearing app data or uninstalling the app.

8. Security

Classic Calculator is designed to work on-device without sending calculator input or results to remote servers. This reduces exposure of user calculations through remote storage or transfer.

9. Children

Classic Calculator is a general-audience app and is not specifically directed to children under 13.

10. Changes to this policy

This Privacy Policy may be updated from time to time. The latest version will be made available at the published policy URL.
