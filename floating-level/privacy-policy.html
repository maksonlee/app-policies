Floating Level Privacy Policy

Effective date: March 24, 2026

Floating Level is developed by Makson Lee. This Privacy Policy explains how Floating Level handles user data.

1. Core functionality

Floating Level displays a spirit level overlay and uses on-device motion sensors to calculate pitch and roll values in real time.

2. Local processing

Sensor readings and overlay rendering are processed locally on the user’s device. Floating Level does not upload sensor data, screenshots, or overlay content to developer-controlled servers.

3. Data collection

Floating Level does not require user accounts and does not collect personal information such as name, email address, phone number, precise location, or payment information.

Floating Level does not collect analytics tied to a user identity through developer-controlled servers.

4. Permissions and device access

Floating Level may request access to the following device capabilities:

  • Draw over other apps, so the floating spirit level can remain visible above other applications
  • Motion sensors, so the app can compute tilt and show bubble movement accurately
  • Foreground service capability, so the overlay can continue running while the user switches apps

These capabilities are used only to provide the app’s core overlay functionality.

5. Data sharing

Floating Level does not sell, rent, or share personal data with third parties.

The app may rely on standard Android and Google Play platform components to run on the device, but Floating Level does not transmit user sensor readings or overlay usage data to developer-operated services.

6. Data retention and deletion

Because processing occurs locally, Floating Level does not retain user data on developer-controlled servers.

If the app stores any temporary operational state on the device, that data remains on the user’s device and can be removed by clearing app data or uninstalling the app.

7. Security

Floating Level is designed to work on-device without sending user data to remote servers. This reduces exposure of sensor or usage data through remote storage or transfer.

8. Children

Floating Level is a general-audience app and is not specifically directed to children under 13.

9. Changes to this policy

This Privacy Policy may be updated from time to time. The latest version will be made available at the published policy URL.
